Logaritmos

Os logaritmos surgiram na necessidade de fazer com que operações com multiplicação e divisão se tornassem mais simples. Diante disso, John Napier criou os logaritmos, baseando-se nas propriedades de multiplicação de potências de mesma base. São usados principalmente na matemática financeira.


Como funcionam os logaritmos?

Como já vimos, a ideia de logaritmos surgiu a partir das potências e suas propriedades, sendo assim, podemos perceber a ligação entre os logaritmos e as equações exponenciais. Os logaritmos se definem formalmente pela fórmula logab=x, onde a é a base, b é o logaritmando e x é o logaritmo. Para o logaritmo existir, deve-se seguir os seguintes critérios: b > 0, a > 0 e a ≠ 1.

Exemplo:

log264 = x

2x = 64

2x = 26 (bases iguais se anulam)

x = 6

Analisando a equação, podemos dizer que o resultado de um logaritmo será um expoente de uma potência, já que 26 = 64.

Casos particulares

Não é sempre que vamos poder obter algum resultado em logaritmos, já que depende dos valores da equação. Desse modo, vemos algumas observações de casos particulares:

  1. A base não pode ser zero, nem 1, nem negativa.
  2. O logaritmando não pode ser 0 nem número negativo.
  3. Caso o logaritmando seja igual a um, o logaritmo dessa base será 0, já que qualquer número elevado a 0 resulta em 1 (Ex. -> log51 = 0, pois 50 = 1).
  4. Sempre que o logaritmo for igual a 1, a base e o logaritmando serão iguais já que qualquer número elevado a 1 será ele mesmo (Ex. -> log55 = 1)

Propriedades operatórias dos Logaritmos

As propriedades dos logaritmos foram criadas para caso a aplicação da simples definição deles não seja suficiente para a resolução. Essas propriedades são extremamente úteis para a resolução de problemas utilizando logaritmos.


1a propriedade Logaritmo de um produto loga(B.C) = logaB + logaC
2a propriedade Logaritmo de um quociente loga(B/C) = logaB - logaC
3a propriedade Logaritmo de uma potência logaBC = C.logaB
logaC√2 = 1/ClogaB
4a propriedade Mudança de base logBC = logaC/logaB

Exemplos (todos os exemplos, com exceção do 40 que foi retirado do material da prof, foram retirados do site Toda Matéria)

1a propriedade - Considerando log 2 = 0,3 e log 3 = 0,48, determine o valor do log 60.

log 60 = log (2.3.10)

log 60 = log 2 + log 3 + log 10

log 60 = 0,3 + 0,48 + 1 = 1,78


2a propriedade - Considerando log 5 = 0,70, determine o valor do log 0,5.

log 0,5 = log(5/10)

log 0,5 = log 5 - log10

log 0,5 = 0,7 - 1

log 0,5 = -0,3


3a propriedade - Considerando log 3 = 0,48, determine o valor do log 81.

log 81 = log 34

log 81 = 4 . log 3

log 81 = 4 . 0,48

log 81 = 1,92


4a propriedade:

log24

log44/log42

1/(1/2)

1.2 = 2


Expressões logarítmicas

Para solucionarmos uma expressão logarítmica, devemos seguir os seguintes passos:

  1. Analisar a condição de existência
  2. Utilizar as propriedades para fazer com que a expressão vire uma expressão exponencial
  3. Analisar o resultado para ver se ele entra nas regras de condição de existência

Exemplos:

log2(x-4) = 3

x-4 = 23

x = 8 + 4

x = 12


log2 + log(x+1) - logx = 1

log[2(x+1)/x] = 1

2(x + 1)/x = 101

2x + 2 = 10x

8x = 2

x = 2/8 = 1/4


Problemas matemáticos

(UERJ - 2016) Admita que a ordem de grandeza de uma medida x é uma potência de base 10, com expoente n inteiro, para 10n - 1/2 <= x <=10n + 1/2. Considere que um terremoto tenha liberado uma energia E, em joules, cujo valor numérico é tal que log10 E = 15,3. A ordem de grandeza de E, em joules, equivale a:

a) 1014

b) 1015

c) 1016

d) 1017


Aplicando a definição de logaritmos:

log E = 15,3

E = 1015,3

Como já dito no problema, a potência deve seguir a seguinte regra:

10n - 1/2 <= x <=10n + 1/2

Calculando, obtemos os seguintes resultados: 1014,5 e 1015,5

De acordo com o enunciado, o expoente da grandeza deve estar entre os dois resultados que encontramos, assim podemos afirmar que o expoente é 15.

Resposta: letra b) 1015
