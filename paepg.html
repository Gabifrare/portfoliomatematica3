Progressão Aritmética e Geométrica

Progressão Aritmética (PA) e Progressão Geométrica (PG) são sequências numéricas, onde na PA existe uma diferença constante entre os termos e na PG existe uma razão constante.

Sequências Numéricas

Antes de entrarmos no conteúdo de PA e PG, é necessário que façamos uma breve recapitulação do conteúdo de sequências numéricas. Uma sequência numérica é caracterizada por uma sequência de números que seguem um padrão. Por exemplo, a sequência dos números pares é uma sequência numérica pois todos os números pois crescem de dois em dois. Também existem outras sequências como os números ímpares e a sequência de Fibonacci.

Progressão Aritmética

Antes de entrarmos no conteúdo de PA e PG, é necessário que façamos uma breve recapitulação do conteúdo de sequências numéricas. Uma sequência numérica é caracterizada por uma sequência de números que seguem um padrão. Por exemplo, a sequência dos números pares é uma sequência numérica pois todos os números pois crescem de dois em dois. Também existem outras sequências como os números ímpares e a sequência de Fibonacci.


Exemplos:

  • (1, 2, 3, 4, 5, 6…)
    • Razão: 1; Classificação: crescente
  • (4, 4, 4, 4, 4, 4…)
    • Razão: 0; Classificação: constante
  • (0, -2, -4, -6, -8…)
    • Razão: -2; Classificação: decrescente

Classificações

  1. Crescente: o termo é maior do que o anterior (r > 0);
  2. Constante: cada termo é igual ao anterior (r = 0);
  3. Decrescente: o termo é menor do que o anterior (r < 0)

Fórmula do Termo Geral

Visto que:

  • a1
  • a2 = a1 + r
  • a3 = a1 + r + r = a1 + 2r
    • ...

Podemos estabelecer a seguinte fórmula: an = a1 + (n - 1)r


    Onde:
  • a1 = primeiro termo
  • an = termo geral
  • n = número de termos
  • r = razão

Qual o número de termos de uma PA de razão -3 cujos primeiro e último termo são, respectivamente, 5 e -22.

Relação da PA com Funções Afim

Como acabamos de ver, a progressão aritmética segue uma sequência onde a razão é igual, ou seja, é igual para todos os termos. Deste modo, é possível fazer um paralelo com as funções afim, já que ambas têm essa característica em comum.

Na PA, não podemos traçar uma reta entre os pontos, já que não existem números reais entre eles, enquanto na função afim existe.

Para fazermos a soma de todos os termos de uma PA finita, não precisamos necessariamente somar todos os termos um por um, podemos usar a seguinte fórmula:

Progressão Geométrica

Chama-se de Progressão Geométrica quando um termo de uma sequência numérica é o resultado do termo anterior multiplicado por um quociente ( q ), que é único para toda a progressão.

Exemplos:

  • (1, 3, 9, 27, 81, …)
    • Quociente: 3; Classificação: crescente
  • (-1, -4, -16, -64, …)
    • Quociente: 4; Classificação: decrescente
  • (5, 5, 5, 5, 5, …)
    • Quociente: 1; Classificação: constante
  • (2, -4, 8, -16, 32, -64, …)
    • Quociente: -2; Classificação: alternante
  • (4, 0, 0, 0, 0, 0, …)
    • Quociente: 0; Classificação: estacionária

Classificações

  1. Crescente: o termo é maior do que o anterior (q > 0);
    • PG com termos positivos: q > 1.
    • PG com termos negativos: 0 < q < 1.
  2. Constante: cada termo é igual ao anterior (q = 0);
    • PG com todos os termos nulos: a1 = 0 e q qualquer.
    • PG com termos iguais e não nulos: a ≠ 0 e q = 1.
  3. Decrescente: o termo é menor do que o anterior (q < 0);
    • PG com termos positivos: 0 < q < 1.
    • PG com termos negativos: q > 1.
  4. Alternante: quando cada termo tem o sinal oposto do anterior (q < 0);
  5. Estacionárias: apenas o primeiro termo é diferente de 0 (a1 ≠ 0 e q = 0).

Fórmula do Termo Geral

Assim como na PA, podemos ver que:

  • a1
  • a2 = a1 . q
  • a3 = a1 . q . q = a1 . q2
  • a4 = a1 . q . q . q = a1 + q3
    • ...

Deste modo, podemos estabelecer a fórmula do termo geral como: an = a1 ∙ qn - 1


    Onde:
  • a1 = primeiro termo
  • an = termo geral
  • n = número de termos
  • q = quociente

Exemplo (retirado do material da professora):

Calcule o 1º termo de uma PG em que a4 = 375 e q = 5.

Soma dos termos de uma PG finita

Do mesmo modo que não precisamos somar cada um dos termos de uma PA, também é possível utilizar uma fórmula em uma PG para fazer a soma. Para isso, usamos a seguinte fórmula:

Sn = a1 ∙ ((1 - qn)/(1 - q))

Soma dos termos de uma PG infinita

Podemos fazer a soma dos termos de uma PG infinita quando -1 < q < 1. Calculamos a soma pela seguinte fórmula:

Sn = a1/(1 - q)
