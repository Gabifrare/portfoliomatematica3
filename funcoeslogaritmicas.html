Funções Logarítmicas

A função Logarítmica é a função inversa da exponencial, dada pela seguinte fórmula: f(x) = logba.

Sendo b > 0, a > 0 e a diferente de 1.

Relação entre funções logarítmicas e exponenciais

Como já dito antes, as funções logarítmicas são o inverso das funções exponenciais. Por exemplo, enquanto uma função exponencial 24 leva ao resultado 16, a função log216 resulta em 4.

Os gráficos das funções são simétricos, como podemos observar na figura abaixo.

Gráfico da Função Logarítmica

Usando como exemplo a função f(x) = log2(x + 2)

1° passo: analisar o crescimento/decrescimento

Para sabermos se a função é crescente ou decrescente, basta verificar o valor da base da função.


base > 1 = crescente

0 < base < 1=decrescente


Exemplos de função crescente e decrescente


2° passo: determinar o domínio

Como já vimos no conteúdo de logaritmos, não são todos os números que podem estar dentro da fórmula. Por esse motivo, devemos analisar o domínio já que ele facilita o processo do gráfico mostrando seu limite. Para descobrirmos o domínio, igualamos o logaritmando a 0.

log2(x + 2)

x + 2 = 0

x = -2


3° passo: encontrar as raízes em x

Assim como nas outras funções, quando o gráfico cruzar o eixo x, devemos encontrar os pontos. Para isso, devemos igualar toda a equação a 0.

log2(x + 2) = 0

x + 2 =20

x + 2 = 1

x = -1


4° passo: interseção com o eixo y

Não são todas as funções que passam pelo eixo y, logo, esse passo apenas é necessário se o domínio for menor que 0. Como no exemplo o domínio é -2, vamos fazer esse passo.

Para encontrarmos a interseção com o eixo y, basta igualar x a 0 e resolver a equação.

log2(0 + 2) = y

log2(2) = y

log22 = 1


5º passo: método da tabela

Para achar outros pontos do gráfico, também podemos montar uma tabela para ajudar no processo.

valores x valores y
-1 log2(-1 + 2) = log2(1) = 0
0 log2(0 + 2) = 1
1 log2(1 + 2) = log2(3) = 1,58
2 log2(2 + 2) = log2(4) = 2
